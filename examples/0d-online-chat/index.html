How it works

Instructions:

  1. Click "Create Room" to generate a room ID
  2. Share the room ID with someone
  3. They enter the room ID and click "Join"
  4. Start chatting directly peer-to-peer!

Note: This demo uses a simple signaling mechanism. In production, you'd use a proper signaling server.
